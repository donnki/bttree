# BtTree
A behaviour tree framework with editor.
![](debug.png)

## Features:
* simple
* fast
* extendable
* visual-editing
* visual-debuggable
* platform-independent

![](editor.png)
![](unityeditor.png)

## Support Languages:
* Lua
* to be continued...

## Usage:
* edit behaviour scripts(json) with editor
* see: lua/src/test.lua as a demo
